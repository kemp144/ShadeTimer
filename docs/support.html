Support

Thank you for using ShadeTimer.

Bug Reports and Feature Requests

For all bug reports, feature requests, and other support inquiries, please use the "Issues" section of our GitHub repository.

This allows us to track all feedback in one place and allows other users to see the status of their inquiries.

Please provide as much detail as possible when submitting an issue, including your macOS version, app version, and steps to reproduce the problem.

Contact

For other questions, you can contact us via GitHub.
